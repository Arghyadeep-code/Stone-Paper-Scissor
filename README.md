# Stone Paper Scissor Game 🎮

A simple Stone Paper Scissor game where you can play against the computer!

## Description
This is a basic Stone Paper Scissor game built with Python. Challenge the computer and see if you can win!

## How to Run

### Option 1: Run Locally
1. **Clone the repository:**
   ```bash
   git clone https://github.com/Arghyadeep-code/Stone-Paper-Scissor.git
   cd Stone-Paper-Scissor
   ```

2. **Run the game:**
   ```bash
   python game.py
   ```

### Option 2: Run on GitHub Codespaces (No Installation Needed!)
1. Click the green `<> Code` button on this repository
2. Go to the **Codespaces** tab
3. Click **Create codespace on main**
4. Once the environment loads, open the terminal and run:
   ```bash
   python game.py
   ```

## Requirements
- Python 3.x installed on your system (for local setup)

## How to Play
1. Run the script
2. Enter your choice when prompted: **Stone**, **Paper**, or **Scissor** (with capital first letter)
3. The computer will make its choice
4. The winner is determined by the classic Stone-Paper-Scissor rules

## Game Rules
- ✂️ **Scissor beats Paper**
- 📄 **Paper beats Stone**
- 🪨 **Stone beats Scissor**
- If both choose the same, it's a **Draw**
