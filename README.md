# Stone Paper Scissor Game

A simple and fun command-line Stone Paper Scissor game where you can play against the computer!

## Overview

This is a Python-based implementation of the classic Stone Paper Scissor game. Test your luck and strategy against the computer opponent with a user-friendly interface.

## Features

- 🎮 **Play Against Computer**: Challenge the computer to rounds of Stone Paper Scissor
- 🎯 **Simple Interface**: Easy-to-use command-line interface
- 📊 **Game Logic**: Fair and random computer moves
- 🔄 **Multiple Rounds**: Play as many rounds as you want

## Requirements

- Python 3.x

## Installation

1. Clone the repository:
```bash
git clone https://github.com/Arghyadeep-code/Stone-Paper-Scissor.git
cd Stone-Paper-Scissor
```

2. No additional dependencies required - uses only Python standard library!

## How to Play

1. Run the game:
```bash
python game.py
```

2. Follow the on-screen prompts to make your choice:
   - Enter **Stone** (or Rock)
   - Enter **Paper**
   - Enter **Scissor**

3. The computer will make its choice
4. The winner is determined by the classic rules:
   - **Stone beats Scissor**
   - **Scissor beats Paper**
   - **Paper beats Stone**

## Game Rules

- Each round, you and the computer make a choice
- The rules are applied to determine the winner
- You can play multiple rounds in succession

## Example Gameplay

```
Welcome to Stone Paper Scissor Game!
Make your choice: Stone, Paper, or Scissor?
Enter your choice: Stone
Computer chose: Paper
You lose! Paper beats Stone.

Play again? (yes/no): yes
```
